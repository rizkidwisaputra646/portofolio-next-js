## Overview
A scalable CRM system for a global enterprise, built using a serverless-first approach on AWS.

## Architecture
- **API:** AWS Lambda with Go 1.21.
- **Database:** Amazon DynamoDB for global replication.
- **Infrastructure:** Managed via Terraform for consistency across regions.
